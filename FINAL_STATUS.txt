## Build Commands

### Android
```bash

```

### iOS
```bash

# Or open in Xcode
```

### Functions (Local Testing)
```bash
cd functions
npm run build          # Compile TypeScript
firebase emulators:start --only functions
```

### Deploy to Firebase
```bash
# Rules only (Free tier)
firebase deploy --only firestore:rules

# With functions (requires Blaze plan)
firebase deploy --only firestore:rules,functions
```

## Firebase Setup Required

1. Create Firebase project at https://console.firebase.google.com
2. Download configs:
   - Android: `google-services.json` â†’ `android/app/`
   - iOS: `GoogleService-Info.plist` â†’ `ios/Runner/` (via FlutterFire)
3. Enable Authentication (Email link sign-in)
4. Create Firestore database
5. Deploy rules: `firebase deploy --only firestore:rules`
6. Seed test data: `cd scripts && node seed_users.js`
