# Security Policy

## Supported Versions

Currently, only the latest version of PSGMX is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability within this project, please report it by opening a private security advisory on GitHub or by contacting the maintainers directly through the repository's issue tracker (please do not disclose details publicly until we have had a chance to respond).

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary.
